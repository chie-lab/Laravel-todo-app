# コントローラー

## 概要

リクエストの処理ロジックをまとめるクラス。`app/Http/Controllers/` に配置する。

## 作成

```bash
php artisan make:controller TaskController
```

リソースコントローラー（CRUD メソッドを自動生成）:

```bash
php artisan make:controller TaskController --resource
```

## 基本的な構造

```php
<?php

namespace App\Http\Controllers;

use App\Models\Task;
use Illuminate\Http\Request;

class TaskController extends Controller
{
    // 一覧表示
    public function index()
    {
        $tasks = Task::all();
        return view('tasks.index', compact('tasks'));
    }

    // 作成フォーム表示
    public function create()
    {
        return view('tasks.create');
    }

    // 保存
    public function store(Request $request)
    {
        $request->validate([
            'title' => 'required|max:255',
        ]);

        Task::create($request->only('title'));

        return redirect()->route('tasks.index');
    }

    // 詳細表示
    public function show(Task $task)
    {
        return view('tasks.show', compact('task'));
    }

    // 編集フォーム表示
    public function edit(Task $task)
    {
        return view('tasks.edit', compact('task'));
    }

    // 更新
    public function update(Request $request, Task $task)
    {
        $request->validate([
            'title' => 'required|max:255',
        ]);

        $task->update($request->only('title'));

        return redirect()->route('tasks.index');
    }

    // 削除
    public function destroy(Task $task)
    {
        $task->delete();

        return redirect()->route('tasks.index');
    }
}
```

## バリデーション

`$request->validate()` でバリデーションを行い、失敗時は自動的に前の画面にリダイレクトする。

```php
$request->validate([
    'title'   => 'required|max:255',
    'body'    => 'nullable|string',
    'due_at'  => 'nullable|date',
]);
```

主なルール:

| ルール | 意味 |
|---|---|
| `required` | 必須 |
| `nullable` | null 許可 |
| `string` | 文字列 |
| `max:255` | 最大255文字 |
| `min:1` | 最小1文字 |
| `integer` | 整数 |
| `email` | メール形式 |
| `date` | 日付形式 |
| `unique:テーブル名` | DB で一意 |

## レスポンス

```php
// ビューを返す
return view('tasks.index', compact('tasks'));

// リダイレクト
return redirect()->route('tasks.index');

// リダイレクト（フラッシュメッセージ付き）
return redirect()->route('tasks.index')->with('success', '保存しました');

// JSON レスポンス（API 用）
return response()->json(['status' => 'ok']);
```

## モデルバインディング

ルートパラメータに対応するモデルを自動で取得する。見つからない場合は 404 を返す。

```php
// ルート定義
Route::get('/tasks/{task}', [TaskController::class, 'show']);

// コントローラー（$task は自動的に Task::find($id) される）
public function show(Task $task)
{
    return view('tasks.show', compact('task'));
}
```
