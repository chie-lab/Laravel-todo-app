# ルーティング

## 基本

ルートは `routes/web.php` に定義する。

```php
use Illuminate\Support\Facades\Route;

Route::get('/hello', function () {
    return 'Hello World';
});
```

## HTTP メソッド

```php
Route::get('/tasks', [TaskController::class, 'index']);
Route::post('/tasks', [TaskController::class, 'store']);
Route::put('/tasks/{id}', [TaskController::class, 'update']);
Route::delete('/tasks/{id}', [TaskController::class, 'destroy']);
```

## URL パラメータ

```php
// 必須パラメータ
Route::get('/tasks/{id}', function (string $id) {
    return 'Task ID: ' . $id;
});

// 任意パラメータ
Route::get('/tasks/{id?}', function (?string $id = null) {
    return $id ?? '一覧';
});
```

## 名前付きルート

ルートに名前をつけると、URL を直書きせずに参照できる。

```php
Route::get('/tasks', [TaskController::class, 'index'])->name('tasks.index');
```

```php
// コントローラーやBladeで使用
$url = route('tasks.index');
```

## リソースルート

CRUD 操作に必要な7つのルートをまとめて定義できる。

```php
Route::resource('tasks', TaskController::class);
```

| メソッド | URI | アクション | ルート名 |
|---|---|---|---|
| GET | /tasks | index | tasks.index |
| GET | /tasks/create | create | tasks.create |
| POST | /tasks | store | tasks.store |
| GET | /tasks/{task} | show | tasks.show |
| GET | /tasks/{task}/edit | edit | tasks.edit |
| PUT/PATCH | /tasks/{task} | update | tasks.update |
| DELETE | /tasks/{task} | destroy | tasks.destroy |

## ルートグループ

共通のミドルウェアやプレフィックスをまとめて適用できる。

```php
Route::middleware(['auth'])->group(function () {
    Route::get('/dashboard', [DashboardController::class, 'index']);
    Route::resource('tasks', TaskController::class);
});
```

```php
Route::prefix('admin')->group(function () {
    Route::get('/users', [AdminController::class, 'index']);
});
// → /admin/users
```
