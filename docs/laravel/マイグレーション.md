# マイグレーション

## 概要

DB テーブルの構造をコードで管理する仕組み。変更履歴をファイルとして残せるため、チーム開発や環境の再構築が容易になる。

## マイグレーションファイルの作成

```bash
php artisan make:migration create_tasks_table
```

既存テーブルへのカラム追加:

```bash
php artisan make:migration add_is_done_to_tasks_table
```

ファイルは `database/migrations/` に作成される。

## マイグレーションファイルの構造

```php
<?php

use Illuminate\Database\Migrations\Migration;
use Illuminate\Database\Schema\Blueprint;
use Illuminate\Support\Facades\Schema;

return new class extends Migration
{
    // マイグレーション実行時（up）
    public function up(): void
    {
        Schema::create('tasks', function (Blueprint $table) {
            $table->id();
            $table->string('title');
            $table->text('description')->nullable();
            $table->boolean('is_done')->default(false);
            $table->timestamps(); // created_at, updated_at
        });
    }

    // ロールバック時（down）
    public function down(): void
    {
        Schema::dropIfExists('tasks');
    }
};
```

## 主なカラム型

| メソッド | 型 | 説明 |
|---|---|---|
| `$table->id()` | BIGINT | 主キー（AUTO INCREMENT） |
| `$table->string('name')` | VARCHAR(255) | 文字列 |
| `$table->text('body')` | TEXT | 長文 |
| `$table->integer('count')` | INT | 整数 |
| `$table->boolean('flag')` | TINYINT | 真偽値 |
| `$table->date('date')` | DATE | 日付 |
| `$table->timestamp('at')` | TIMESTAMP | 日時 |
| `$table->timestamps()` | - | created_at / updated_at を追加 |
| `$table->softDeletes()` | - | deleted_at を追加 |
| `$table->foreignId('user_id')` | BIGINT | 外部キー |

## カラム修飾子

```php
$table->string('title')->nullable();         // NULL 許可
$table->string('status')->default('pending'); // デフォルト値
$table->string('email')->unique();            // ユニーク制約
$table->text('body')->nullable()->after('title'); // 位置指定
```

## 外部キー制約

```php
$table->foreignId('user_id')->constrained()->cascadeOnDelete();
// → users テーブルの id を参照
//   ユーザー削除時にタスクも削除（CASCADE）
```

## マイグレーションコマンド

```bash
# 実行（未適用のファイルをすべて適用）
php artisan migrate

# ロールバック（直前のバッチを取り消し）
php artisan migrate:rollback

# 全テーブルを削除して再実行
php artisan migrate:fresh

# シーダーも同時実行
php artisan migrate:fresh --seed

# 状態確認
php artisan migrate:status
```

## シーダー（初期データ投入）

```bash
php artisan make:seeder TaskSeeder
```

```php
// database/seeders/TaskSeeder.php
public function run(): void
{
    Task::create(['title' => 'サンプルタスク', 'is_done' => false]);
}
```

```bash
php artisan db:seed
php artisan db:seed --class=TaskSeeder
```
