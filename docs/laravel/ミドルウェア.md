# ミドルウェア

## 概要

HTTP リクエストとレスポンスの間に割り込む処理。認証チェック・ログ記録・CORS ヘッダー付与など、コントローラーに到達する前後の共通処理を担う。

```
リクエスト → [ミドルウェア1] → [ミドルウェア2] → コントローラー
                                                       ↓
レスポンス ← [ミドルウェア1] ← [ミドルウェア2] ←  処理結果
```

## 作成

```bash
php artisan make:middleware EnsureTaskOwner
```

`app/Http/Middleware/` に作成される。

## 基本構造

```php
<?php

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;

class EnsureTaskOwner
{
    public function handle(Request $request, Closure $next): Response
    {
        // コントローラーの前処理
        if ($request->task->user_id !== $request->user()->id) {
            abort(403);
        }

        $response = $next($request); // コントローラーへ処理を渡す

        // コントローラーの後処理（必要な場合）

        return $response;
    }
}
```

## ルートへの適用

```php
// 単一ルートに適用
Route::get('/tasks/{task}', [TaskController::class, 'show'])
    ->middleware(EnsureTaskOwner::class);

// グループに適用
Route::middleware(['auth', EnsureTaskOwner::class])->group(function () {
    Route::resource('tasks', TaskController::class);
});
```

## Laravel 標準ミドルウェア

| ミドルウェア | 役割 |
|---|---|
| `auth` | ログイン済みかチェック（未認証なら login にリダイレクト） |
| `guest` | 未ログインかチェック（認証済みなら home にリダイレクト） |
| `verified` | メール認証済みかチェック |
| `throttle:60,1` | レート制限（1分間に60回まで） |

## グローバルミドルウェア

`bootstrap/app.php` でアプリ全体に適用できる。

```php
->withMiddleware(function (Middleware $middleware) {
    $middleware->append(MyGlobalMiddleware::class);
})
```

## ミドルウェアのエイリアス

```php
->withMiddleware(function (Middleware $middleware) {
    $middleware->alias([
        'task.owner' => EnsureTaskOwner::class,
    ]);
})
```

```php
Route::get('/tasks/{task}', [TaskController::class, 'show'])
    ->middleware('task.owner');
```
